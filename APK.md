# APK do Base3

Este projeto agora está preparado para gerar um APK Android usando Capacitor.

## Como gerar

1. Envie as alterações para o GitHub.
2. Abra a aba `Actions` do repositório.
3. Execute o fluxo `Build Android APK`.
4. Quando terminar, baixe o artefato `base3-debug-apk`.

## O que esse APK faz

- Abre o sistema online em `https://base3.onrender.com`
- Mantém o backend no Render
- Funciona como um app Android instalável

## Observação

O build gerado aqui é um APK de teste, útil para instalar no celular e validar o sistema.
Se você quiser, depois eu posso deixar uma versão de release assinada.
